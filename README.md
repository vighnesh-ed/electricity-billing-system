# ⚡ Electricity Billing System

A responsive Electricity Billing System developed using PHP, MySQL, Bootstrap, HTML, and CSS.

## Features

- Responsive user interface
- Electricity bill generation
- Slab-wise bill calculation
- Automatic bill amount calculation
- Stores bills in MySQL database
- Printable bill (PDF support)
- Bill history (if implemented)

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- XAMPP

---

## Slab Rates

| Units | Rate |
|--------|------|
| 0-50 | ₹3.50 |
| 51-100 | ₹4.00 |
| 101-250 | ₹5.20 |
| Above 250 | ₹6.50 |

---

## Installation

1. Install XAMPP.
2. Copy the project into the `htdocs` folder.
3. Start Apache and MySQL.
4. Import `database.sql` into phpMyAdmin.
5. Open:

http://localhost/electricity-billing-system/
